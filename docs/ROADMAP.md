# PMAgent 로드맵

이 문서는 PMAgent 프로젝트의 향후 개발 계획과 방향성을 설명합니다.

## 현재 버전: 0.1.0

현재 버전에서는 다음과 같은 기본 기능을 제공합니다:

- 프로젝트 관리 (생성, 조회, 업데이트, 삭제)
- 태스크 관리 (생성, 조회, 업데이트, 삭제)
- RESTful API
- 파이썬 클라이언트 라이브러리 (동기 및 비동기 지원)
- 명령행 인터페이스 (CLI)
